adb命令小记

1、如何打开adb调试？如何打开开发者模式？如何进入工厂菜单？进入工厂菜单的方式有什么途径？

遥控器 debugmenu 进入Android set 全部打开

2、如何配置adb环境变量？

google

3、使用什么adb命令连接整机进行调试

adb connect ip:5555

4、使用什么adb命令截图

adb shell screencap -p /sdcard/sdkf.jpg

5、使用什么adb命令将整机系统的一个文件拉取到电脑

adb pull /sdcard/sdkf.jpg /Users/lei/downloads

6、使用什么adb命令将电脑的一个文件推送到整机系统

adb push /Users/lei/Downloads/s1.png /sdcard

7、使用什么adb命令查看当前界面是哪个应用

adb shell dumpsys window | grep mCurrentFocus

8、使用什么adb命令打开某个Activity

Activity Manager(am)工具 shell 发送远程内核命令
adb shell am start -n com.example.myapplication/.main.MainActivity

9、使用什么adb命令发送某个广播

同8 startservice

10、使用什么adb命令打开某个服务

[–user user_id | all | current]：指定要发送到的用户；如果未指定，则发送到所有用户
同8 broadcast

11、使用什么adb命令获取某个应用的版本信息

adb shell dumpsys package com.example.myapplication | grep version

12、使用什么adb命令安装应用

install [options] path 将软件包（通过 path 指定）安装到系统。
选项：
-l：安装具有转发锁定功能的软件包。
-r：重新安装现有应用，保留其数据。
-t：允许安装测试 APK。
-i installer_package_name：指定安装程序软件包名称。
-s：在共享的大容量存储（如 sdcard）上安装软件包。
-f：在内部系统内存上安装软件包。
-d：允许版本代码降级。
-g：授予应用清单中列出的所有权限。
adb install -r apk路径

13、使用什么adb命令卸载应用

-k：移除软件包后保留数据和缓存目录。
adb uninstall com.example.myapplication

14、使用什么adb命令获取某个系统属性值

adb shell getprop [key]
adb shell getprop 不加全显示

15、使用什么adb命令设置某个系统属性值

adb shell setprop [key] [value]

16、如何通过adb获得系统的ANR文件和log文件

adb root
adb pull /data/anr /Users/lei/Downloads

17、如何查看系统CPU占用率

adb shell top

18、如何查看某个应用的进程号？如何杀死某个应用？

adb root
adb shell ps 进程列表
adb shell kill [pid]

19、AS配置Markdown插件

markdown navigator

20 卸载系统应用

adb root
adb remount
adb shell
cd /system/app/
re -rf PinyinIME/

21 重启adb

adb -A | grep adb
kill 进程号
或者
adb kill-server 再 adb start-server
